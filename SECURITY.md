# Security Policy

## Supported Versions

This is a static blog hosted on GitHub Pages. The following versions receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| latest (master) | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Contact the maintainer directly via [LinkedIn](https://www.linkedin.com/in/pschmidt3010) or email.
3. Include a clear description of the vulnerability, steps to reproduce, and potential impact.

You can expect an acknowledgement within **48 hours** and a resolution or mitigation plan within **7 days** for valid reports.

## Scope

This repository covers the static HTML/CSS/JS source for [endlesstech.blog](https://endlesstech.blog). In-scope items include:

- Cross-site scripting (XSS) via blog content
- Malicious third-party resource injection
- Sensitive data exposure in source files

Out-of-scope items:

- GitHub Pages infrastructure itself
- Theoretical issues without practical impact
- Issues in external services linked from the blog
